# Galéria Platform - Specifikáció

## Funkcionális követelmények
1. **Felhasználókezelés**
   - Regisztráció (email, felhasználónév, jelszó)
   - Bejelentkezés
   - Profiloldal (saját feltöltések, adatok módosítása)

2. **Tartalomkezelés**
   - Kép/videó feltöltés (max. 10MB)
   - Tartalmak böngészése galériában
   - Saját tartalmak törlése

3. **Közösségi funkciók**
   - Posztok like-olása
   - Kommentelés
   - Más felhasználók követése

4. **Felfedezés**
   - Keresés cím és leírás alapján
   - Szűrés kategória szerint
   - "Ajánlott tartalmak" főoldalon

## Nem funkcionális követelmények
- Reszponzív design
- Jelszavak biztonságos tárolása
- REST API kialakítása
- MySQL adatbázis használata
